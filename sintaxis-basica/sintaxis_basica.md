# Sintaxis básica

Python destaca por una sintaxis clara y fácil de leer, lo que lo convierte en un lenguaje ideal tanto para principiantes como para desarrolladores experimentados. Su estructura se basa en la indentación y en palabras clave simples que facilitan la comprensión del flujo del programa.

En esta sección se presentan las **estructuras fundamentales del lenguaje**, necesarias para escribir código ordenado, legible y bien organizado.

---

## Estructuras y palabras clave principales

| Nombre                 | Palabra reservada | Ejemplo de sintaxis                    |
|-----------------------|------------------|----------------------------------------|
| Condicional `if`      | `if`             | `if condición:`                        |
| Condicional `elif`    | `elif`           | `elif otra_condición:`                 |
| Condicional `else`    | `else`           | `else:`                                |
| Bucle `for`           | `for`            | `for elemento in secuencia:`           |
| Bucle `while`         | `while`          | `while condición:`                     |
| Sentencia `break`     | `break`          | `break`                                |
| Sentencia `continue`  | `continue`       | `continue`                             |
| Función               | `def`            | `def nombre(parámetros):`              |
| Clase                 | `class`          | `class NombreClase:`                   |
| Variable global       | `global`         | `global nombre_variable`               |
| Manejo de excepciones | `try`, `except`  | `try: ... except Excepción:`           |
| Bloque `finally`      | `finally`        | `finally:`                             |
| Retorno de valores    | `return`         | `return valor`                         |
| Comprobación de tipo  | `is`             | `if variable is tipo:`                 |
| Pertenencia           | `in`             | `if elemento in colección:`            |
| Importar módulo       | `import`         | `import nombre_módulo`                 |
| Importación selectiva | `from`, `import` | `from módulo import nombre`            |

---

Estas construcciones forman la base de la programación en Python y permiten controlar el flujo del programa, reutilizar código y organizar proyectos de manera eficiente.
