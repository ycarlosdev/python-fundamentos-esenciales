# Módulos externos (parte 1 de 2)

Además de la biblioteca estándar, Python permite instalar **módulos externos** para ampliar sus capacidades. Estas librerías de terceros cubren áreas como **peticiones web**, **análisis de datos**, **visualización**, **machine learning** y **visión artificial**.

Conocer las librerías más utilizadas ayuda a elegir la herramienta adecuada según el tipo de proyecto.

---

## Librerías externas más utilizadas

| Librería | Descripción |
|---------|-------------|
| `requests` | Permite realizar peticiones HTTP de forma sencilla |
| `python-dotenv` | Carga variables de entorno desde archivos `.env` |
| `numpy` | Cálculo numérico y manipulación de arrays multidimensionales |
| `pandas` | Análisis y manipulación de datos mediante DataFrames |
| `matplotlib` | Creación de gráficos y visualizaciones |
| `seaborn` | Extensión de `matplotlib` con estilos avanzados |
| `scipy` | Herramientas de cálculo científico y optimización |
| `scikit-learn` | Algoritmos de Machine Learning (clasificación, regresión, clustering) |
| `tensorflow` | Framework de Machine Learning y Deep Learning de Google |
| `torch` | Biblioteca de Deep Learning desarrollada por Meta (PyTorch) |
| `opencv-python` | Procesamiento de imágenes y visión por computador |
| `pillow` | Manipulación y procesamiento de imágenes |

---

Estas librerías permiten construir aplicaciones más complejas y especializadas, complementando los fundamentos del lenguaje.
