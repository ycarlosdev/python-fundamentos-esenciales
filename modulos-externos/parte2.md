# Módulos externos (parte 2 de 2)

En esta sección se incluyen librerías externas orientadas a **web scraping**, **automatización**, **desarrollo web**, **bases de datos**, **testing**, **CLI** y **aplicaciones multiplataforma**. Estas herramientas permiten llevar Python a entornos de producción reales.

---

## Librerías externas adicionales

| Librería | Descripción |
|---------|-------------|
| `beautifulsoup4` | Extracción de datos desde páginas web (Web Scraping) |
| `selenium` | Automatización de navegación web mediante scripts |
| `fastapi` | Framework moderno y rápido para crear APIs |
| `flask` | Framework ligero para desarrollo web |
| `django` | Framework robusto para aplicaciones web |
| `reflex` | Framework para crear aplicaciones web full-stack |
| `sqlalchemy` | ORM para interactuar con bases de datos SQL |
| `pymongo` | Cliente para bases de datos MongoDB |
| `pytest` | Librería para pruebas unitarias |
| `loguru` | Biblioteca avanzada para logging |
| `typer` | Creación sencilla de aplicaciones de línea de comandos |
| `rich` | Añade colores y estilos a la salida en terminal |
| `kivy` | Framework para aplicaciones gráficas multiplataforma |
| `flet` | Framework para crear aplicaciones Flutter multiplataforma |

---

> Todos estos módulos deben instalarse previamente utilizando:
>
> ```bash
> pip install nombre_del_modulo
> ```

El uso adecuado de estas librerías permite ampliar las capacidades de Python y adaptarlo a prácticamente cualquier tipo de proyecto.
