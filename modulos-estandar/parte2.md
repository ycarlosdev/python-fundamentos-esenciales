# Módulos estándar (parte 2 de 2)

En esta segunda parte se incluyen módulos de la biblioteca estándar orientados a **optimización**, **concurrencia**, **seguridad**, **comunicación**, y **desarrollo de aplicaciones**. Estos módulos permiten construir soluciones más robustas sin depender de librerías externas.

---

## Módulos adicionales

| Módulo | Descripción |
|--------|-------------|
| `functools` | Proporciona funciones de orden superior y utilidades como caché |
| `operator` | Ofrece funciones rápidas para operaciones matemáticas y lógicas |
| `statistics` | Incluye funciones estadísticas como media, mediana y desviación estándar |
| `hashlib` | Permite generar hashes criptográficos (SHA256, MD5, entre otros) |
| `logging` | Facilita el registro de eventos y errores en aplicaciones |
| `argparse` | Simplifica el manejo de argumentos desde la línea de comandos |
| `shutil` | Permite copiar, mover y eliminar archivos y directorios |
| `socket` | Soporta comunicación en red mediante sockets |
| `threading` | Permite ejecutar múltiples hilos de forma concurrente |
| `multiprocessing` | Soporta la ejecución de múltiples procesos en paralelo |
| `subprocess` | Ejecuta comandos del sistema y captura su salida |
| `tkinter` | Biblioteca estándar para crear interfaces gráficas (GUI) |

---

El uso adecuado de estos módulos permite ampliar significativamente las capacidades de Python manteniendo un código limpio y eficiente.
