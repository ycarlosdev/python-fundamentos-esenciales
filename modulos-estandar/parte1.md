# Módulos estándar (parte 1 de 2)

Python incluye una **biblioteca estándar muy amplia**, compuesta por módulos listos para usarse sin necesidad de instalación adicional. Estos módulos permiten resolver tareas comunes como la manipulación de archivos, operaciones matemáticas, manejo de fechas, procesamiento de texto y más.

Conocer y utilizar la biblioteca estándar ayuda a **simplificar el desarrollo**, reducir dependencias externas y escribir código más eficiente y mantenible.

---

## Módulos más utilizados

| Módulo | Descripción |
|--------|-------------|
| `os` | Permite interactuar con el sistema operativo (archivos, directorios y procesos) |
| `sys` | Proporciona acceso a variables y funciones del intérprete de Python |
| `math` | Ofrece funciones matemáticas avanzadas (raíces, logaritmos, trigonometría) |
| `random` | Genera números aleatorios y selecciona elementos al azar |
| `datetime` | Maneja fechas y horas |
| `time` | Funciones para controlar el tiempo y pausas en la ejecución |
| `json` | Permite trabajar con datos en formato JSON (serialización y deserialización) |
| `csv` | Facilita la lectura y escritura de archivos CSV |
| `re` | Proporciona herramientas para trabajar con expresiones regulares |
| `collections` | Incluye estructuras de datos avanzadas como `deque`, `Counter` y `defaultdict` |
| `itertools` | Ofrece utilidades para trabajar con iteradores y combinaciones de datos |

---

Estos módulos cubren una gran variedad de necesidades comunes y forman la base de muchos proyectos en Python.
