# Funciones (parte 1 de 3)

Las funciones son uno de los pilares de Python, ya que permiten **reutilizar código**, mejorar la **legibilidad** y construir programas más **modulares y mantenibles**.  
El lenguaje incluye una amplia colección de funciones integradas que facilitan el trabajo con datos, desde operaciones básicas hasta el manejo de colecciones.

A continuación se muestran algunas de las funciones más utilizadas en Python y su propósito principal.

---

## Funciones integradas comunes

| Función | Descripción |
|-------|-------------|
| `print()` | Muestra texto o valores en la consola |
| `input()` | Permite introducir datos desde la consola |
| `len()` | Devuelve la cantidad de elementos de un objeto |
| `type()` | Indica el tipo de dato de una variable |
| `range()` | Genera una secuencia de números dentro de un rango |
| `int()`, `float()`, `str()` | Convierte valores a entero, flotante o cadena |
| `list()`, `tuple()`, `set()`, `dict()` | Crea listas, tuplas, conjuntos o diccionarios |
| `sum()` | Suma los elementos de un iterable |
| `min()`, `max()` | Obtiene el valor mínimo o máximo de un iterable |
| `sorted()` | Ordena elementos de listas u otros iterables |
| `abs()` | Devuelve el valor absoluto de un número |
| `round()` | Redondea un número a una cantidad específica de decimales |
| `map()` | Aplica una función a cada elemento de un iterable |
| `filter()` | Filtra elementos de un iterable según una condición |
| `reduce()` | Reduce un iterable a un único valor aplicando una función |

---

Estas funciones permiten resolver gran parte de los problemas cotidianos sin necesidad de escribir lógica compleja desde cero, lo que hace a Python un lenguaje práctico y expresivo.
