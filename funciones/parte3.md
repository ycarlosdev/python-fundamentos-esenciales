# Funciones (parte 3 de 3)

En esta última parte se incluyen funciones menos comunes pero igualmente importantes, que permiten trabajar con **ejecución dinámica de código**, **manipulación de objetos**, **introspección**, y **estructuras avanzadas**. Conocerlas ayuda a comprender mejor el funcionamiento interno de Python.

---

## Funciones integradas avanzadas

| Función | Descripción |
|-------|-------------|
| `pow()` | Calcula la potencia de un número |
| `divmod()` | Devuelve el cociente y el resto de una división |
| `copy()` | Crea una copia superficial de un objeto |
| `eval()` | Evalúa una expresión contenida en una cadena |
| `exec()` | Ejecuta un bloque de código definido como texto |
| `format()` | Formatea cadenas utilizando valores personalizados |
| `reversed()` | Invierte el orden de un iterable |
| `slice()` | Crea una porción de un iterable sin modificar el original |
| `callable()` | Comprueba si un objeto puede ser invocado como función |
| `dir()` | Muestra los atributos y métodos disponibles de un objeto |
| `frozenset()` | Crea un conjunto inmutable |
| `complex()` | Genera números complejos |
| `bool()` | Convierte un valor a tipo booleano |
| `super()` | Permite acceder a métodos de la superclase |
| `bin()` | Obtiene la representación binaria de un número |
| `globals()` | Devuelve el diccionario de variables globales |
| `locals()` | Devuelve el diccionario de variables locales |

---

Estas funciones completan el panorama de herramientas que Python ofrece de forma nativa y permiten escribir código más flexible, potente y consciente del entorno de ejecución.
