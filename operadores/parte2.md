# Operadores (parte 2 de 3)

En esta sección se presentan los **operadores lógicos** y los **operadores de asignación**, fundamentales para construir condiciones complejas y modificar valores de forma eficiente durante la ejecución de un programa.

---

## Operadores lógicos

Se utilizan para combinar o negar expresiones booleanas.

| Nombre | Representación | Ejemplo de uso |
|------|----------------|---------------|
| AND  | `and`          | `a and b`     |
| OR   | `or`           | `a or b`      |
| NOT  | `not`          | `not a`       |

Estos operadores son clave en estructuras condicionales y bucles.

---

## Operadores de asignación

Permiten asignar valores a variables, así como modificar su contenido combinando operaciones aritméticas o bitwise.

| Nombre                         | Representación | Ejemplo de uso |
|-------------------------------|----------------|---------------|
| Asignación simple             | `=`            | `x = 5`       |
| Asignación con suma           | `+=`           | `x += 3`      |
| Asignación con resta          | `-=`           | `x -= 3`      |
| Asignación con multiplicación | `*=`           | `x *= 3`      |
| Asignación con división       | `/=`           | `x /= 3`      |
| Asignación con división entera| `//=`          | `x //= 3`     |
| Asignación con módulo         | `%=`           | `x %= 3`      |
| Asignación con potencia       | `**=`          | `x **= 3`     |
| Asignación con bitwise AND    | `&=`           | `x &= 3`      |
| Asignación con bitwise OR     | `|=`           | `x |= 3`      |
| Asignación con bitwise XOR    | `^=`           | `x ^= 3`      |

---

El uso correcto de estos operadores mejora la claridad del código y permite escribir expresiones más compactas y expresivas.
