# Operadores (parte 1 de 3)

Los operadores son elementos fundamentales en Python, ya que permiten realizar **cálculos**, **comparaciones** y **evaluaciones lógicas** dentro de un programa. A través de ellos se manipulan los datos y se toman decisiones en el flujo de ejecución.

En esta primera parte se abordan los **operadores aritméticos** y los **operadores de comparación**, que son los más utilizados en el día a día.

---

## Operadores aritméticos

Se utilizan para realizar operaciones matemáticas básicas entre valores numéricos.

| Nombre                | Representación | Ejemplo de uso |
|----------------------|----------------|---------------|
| Suma                 | `+`            | `a + b`       |
| Resta                | `-`            | `a - b`       |
| Multiplicación       | `*`            | `a * b`       |
| División             | `/`            | `a / b`       |
| División entera      | `//`           | `a // b`      |
| Módulo (residuo)     | `%`            | `a % b`       |
| Exponenciación       | `**`           | `a ** b`      |

---

## Operadores de comparación

Permiten comparar valores y devuelven un resultado booleano (`True` o `False`).

| Nombre            | Representación | Ejemplo de uso |
|------------------|----------------|---------------|
| Igual a          | `==`           | `a == b`      |
| Distinto de      | `!=`           | `a != b`      |
| Mayor que        | `>`            | `a > b`       |
| Menor que        | `<`            | `a < b`       |
| Mayor o igual que| `>=`           | `a >= b`      |
| Menor o igual que| `<=`           | `a <= b`      |

---

Estos operadores son la base para construir expresiones, condiciones y lógica de control en Python. En las siguientes partes se profundizará en otros tipos de operadores.
